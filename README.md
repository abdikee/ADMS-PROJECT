# Student Academic Record Management System (SAMS)

A comprehensive web-based system for managing student academic records, built with React, Node.js, Express, and MySQL.

## 🌟 Features

- **Role-Based Access Control**: Admin, Teacher, and Student roles with specific permissions
- **Student Management**: Complete CRUD operations for student records
- **Teacher Management**: Manage teacher profiles and assignments
- **Subject Management**: Create and manage subjects
- **Marks Entry**: Record and track student marks
- **Report Generation**: Generate academic reports with rankings
- **Credential Management**: Admin can generate login credentials for students/teachers
- **Real-time Database**: MySQL database integration for persistent data storage

## 🏗️ Architecture

### Frontend
- React 18 with TypeScript
- Vite for fast development
- TailwindCSS for styling
- Shadcn/ui components
- React Router for navigation
- Context API for state management

### Backend
- Node.js with Express
- MySQL database
- JWT authentication
- RESTful API design
- Bcrypt for password hashing

### Prerequisites
- Node.js (v18 or higher)
- MySQL 8.x
- npm or yarn

### 1. Install Dependencies

Frontend:
```bash

```

Backend:

## 🔧 Configuration

```env

DB_USER=abdike
```

## 🎯 User Roles & Permissions

### Admin
- Full system access
- Manage students, teachers, subjects
- Generate credentials
- View all reports
- System configuration

### Teacher
- View assigned classes
- Enter marks for students
- View reports for assigned classes
- Update student information

### Student
- View own academic records
- View own marks and reports
- View class schedule
- Access library resources
